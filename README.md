# VoronoiDiagram

Find the settings.json file in the StreamingAssets directory.

JSON property types:

"RandomPoints":(BOOLEAN),

"RandomColors":(BOOLEAN),

"AmountOfRandomPoints":(INT),

"ConeRadius":(FLOAT),

"ConeSegments":(INT),

"DrawVertices":(BOOLEAN),

"ShowSeeds":(BOOLEAN),

"Orthographic":(BOOLEAN),

"Isometric":(BOOLEAN),

"Speed":(FLOAT),

"Shader":(STRING),

"Points": (ARRAY)

"Color":(ARRAY)
